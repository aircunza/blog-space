## ðŸ Getting Started

### Prerequisites

- Docker & Docker Compose installed
- Node.js >= 18
- npm

### Installation

```bash
git clone https://github.com/aircunza/blog-space.git
cd space-blog
npm install
```

### Running the Project

Spin up the databases and start the backend:

```bash
# Start PostgreSQL and pgAdmin
docker-compose up -d

# Start the development server
npm run dev
```

### Running Tests

```bash
npm run test
```
